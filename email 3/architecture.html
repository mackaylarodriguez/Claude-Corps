Volunteer Confirmation Flow

Riverbend Food Alliance — architecture sketch for Priya & Diane

Uses tools we already pay for Needs Diane approval (~$15/mo) Optional — not on critical path

How it works (v1)

Volunteer picks a shift on your page → signup lands in Airtable (one list, not Form + Sheet + Airtable). A Google Apps Script job runs daily, finds tomorrow's shifts, and sends reminder texts. Every Monday morning, Slack posts a one-line health check: reminders sent, open spots, errors.

What is MCP?

Model Context Protocol — a standard way for Claude to connect to outside tools (like Airtable) so it can look up real data instead of guessing. Useful later if you want Claude to answer "who's on Saturday?" or draft cancel/reschedule replies. Not needed for signup or reminders to work.

Cost (for Diane)

New spend: Twilio SMS ~$15/mo (phone number + ~200 texts). Everything else uses Google Workspace, Airtable, and Slack we already pay for. Apps Script and Google Sites are free.

Time saved (for Diane)

Handbook says ~6 hrs/week on form + sheet + hand-texts. Automating reminders recovers ~4 hrs/week (~$500/mo in Priya's time at a conservative $30/hr). Payback on Twilio is < one week.
